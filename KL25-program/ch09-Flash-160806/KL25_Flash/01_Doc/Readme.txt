硬件说明：PORTB19=红灯；PORTB18= 绿灯；PORTB9= 蓝灯；


1. 工程概述
(1)本工程为KL25微控制器无操作系统应用工程
(2)对flash内存进行擦除，写入、读取（按逻辑地址）、读取（按物理地址）操作
(3)对flash内存进行保护操作
(3)对flash内存进行加密、解密操作

2. 软件环境
(1)Kinetis Design Studio IDE v3.0.0.

3. 硬件环境
(1)KL25 SD13-01 EVB板
(2)蓝灯对应引脚为PTB9
(3)串口2发送引脚为PTE22，接收引脚为PTE23
    
4. 实验现象
(1)KL25 SD13-01 EVB板板上的蓝灯闪烁；
(2)利用KL25-Flash-Csharp测试工具进行测试，按照高端界面提示进行相应的读写擦除以及保护的测试；
(3)扇区保护之后，擦除操作‘E’与写入操作‘W’将无法成功。
(4)Flash扇区的保护状态在芯片复位以后将自动解除。
(5)通过更改启动文件startup_MKL25Z4.S中配置域，来对芯片进行加密和去除密码（安全状态和不安全状态），芯片加密之后,程序不能通过CW10.3进行写入，必须进行大规模擦除操作后，使芯片恢复空白片状态。


5. 实验注意
(1)可以通过高端PC KL25-Flash-Csharp来实时接受数据
(2)先打开串口，后开启KL25 SD13-01 EVB板电源
