《kl25_mqxlite》工程说明
时          间：2016-03-16   
版          本：V1.0
作          者：王磊

1. 工程概述
(1)本工程为KL25微控制器MQX-Lite系统应用工程，提供实现基于MQX-Lite中断框架的串口中断服务例程
(2)初始化串口2作为默认通信设备与外界通信，波特率为9600，1位停止位，无校验;
(3)用串口2回发接受到的数据；
(4)在应用任务中使用..\05_Driver\uart中的独立串口驱动实现串口收发数据
(5)5个任务和一个中断服务例程：
    task_main:开总中断，系统自启动任务，打印欢迎信息，创建其它任务
    task_A:闪烁红灯，亮1秒灭2秒
    task_B:闪烁绿灯，亮1秒灭2秒
    task_C:闪烁蓝灯，亮1秒灭2秒
    task_D:将串口2中断接收的完整数据帧发送至PC机
    UART2_RX_ISR:串口2中断服务例程，回发收到的数据
(6)系统使用外部扩展8M有源晶振，工作时钟总线频率为48M
2. 软件环境
(1)KDS 3.0.0
(2)Freescale MQX-Lite 3.0.0

3. 硬件环境
(1)SD-FSL-KL25-EVB评估板
(2)串口转USB接线：白色：发送线(Tx,Transmit);绿色：接收线(Rx,Receive);黑色：地线(GND,Ground)

4. 实验现象
(1)通过串口2打印"light red(green\blue) changed\r\n"
(2)三色灯循环点亮，每种颜色的灯各亮1秒
(3)串口2以中断方式接收，回发接收到的数据

5. 备注
(1)本工程使用的是本实验室开发的独立设备驱动如，GPIO和UART,并使用指示灯应用构件light。
(2)用户自定义的ISR放在isr.h中声明,在isr.c中实现
