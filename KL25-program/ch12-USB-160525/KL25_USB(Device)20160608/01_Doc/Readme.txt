硬件说明：PORTB19=红灯；PORTB18= 绿灯；PORTB9= 蓝灯；


1. 工程概述
(1)本工程为KL25微控制器无操作系统应用工程，USB设备样例程序
(2)PC端使用C#编写的USB测试程序

2. 软件环境
(1)Kinetis Design Studio IDE v3.0.0.

3. 硬件环境
(1)KL25 SD13-01 EVB板
(2)蓝灯对应引脚为PTB9

    
4. 实验现象
(1)KL25 SD13-01 EVB板板上的蓝灯闪烁
(2)PC端程序显示枚举到的USB设备信息
(3)PC端程序发送数据给底端MCU，MCU接收后，保存到全局变量中。
(4)PC机向底端发送要数据命令，MCU通过USB将收到的数据放入发送缓冲区中，PC机程序将数据取走。

5. 修改说明
(1)本程序对收发的帧定义了帧格式，帧第一个自己为帧类型，其余字节为数据，并在includes.h中
定义了这些帧格式，在收到数据后会根据帧类型进行相应处理，如需增加新的帧格式需在头文件中增加
定义，并在main函数循环中进行解帧处理。
(2)在修改时注意需同时修改PC端C#程序，否则可能会导致如法完成所需功能。
