硬件说明：PORTB19=红灯；PORTB18= 绿灯；PORTB9= 蓝灯；


1. 工程概述
(1)本工程为KL25微控制器无操作系统应用工程，中断和UART样例程序
(2)使用串口与外界通信，使用串口1，波特率为9600，1位停止位，无校验
(3)启动串口接收中断，回发接收数据

2. 软件环境
(1)Kinetis Design Studio IDE v3.0.0.

3. 硬件环境
(1)KL25 SD13-01 EVB板
(2)串口2发送引脚为PTE22，接收引脚为PTE23
(3)将两块KL25板子的主机PTC9与从机PTC2、主机PTC8与从机PTC1，另外PTC9与PTC2、PTC8与PTC1分别
         通过电阻（推荐阻值为3.3K）上拉到+5V电源。注意：该驱动在3.3K电阻负载下开发，改
         变其负载可能需要改变MULT与ICR，即改变保持值SDA，否则可能通信不稳	
         
4. 实验现象
(1)按本机的复位键通过串口发送字符串"This is iic Test!"
(2)接收主机发送过来的数据并通过串口1发送到PC机
(3)通过串口发送字符串"Version3.4"至PC机
