
功能概要：理解TSI触摸编程

版本更新：2012-10-12  V1.0，2013-11-12 V2.0（WYH），2014-08-11 V3.0 (转入KDS)
          2016-03-14 V4.0 (更新到CMSIS标准)，2017-04-1 V5.0
硬件说明：PORTB19=红灯；PORTB18= 绿灯；PORTB9= 蓝灯；

1. 工程概述
(1)使用串口与外界通信，使用串口1，波特率为9600，1位停止位，无校验
(2)启动串口接收中断，回发接收数据，实现触摸控制计数功能

2. 软件环境:KDS v3.0.0.

3. 硬件环境与连接
(1)SD-FSL-KL25-EVB板
(2)蓝灯对应引脚为PTB9,红灯对应引脚为PTB19
(3)串口1发送引脚为PTE0，接收引脚为PTE1
(4)触摸片与触摸引脚为PTA4连接

4. 实验现象
(1)程序下载成功后，SD-FSL-KL25-EVB板板上的蓝灯闪烁。
(2)通过串口1发送字符串"This is TSI Test"
(3)当产生TSI中断时,蓝灯熄灭，点亮红色小灯,并把触摸次数通过串口1发送到PC机，
   某次触摸时间较长时，会通过串口1按照一定的时间间隔输出"你已长按，请松手!"的提示。

5. 注意事项
   在多次触摸触摸片时，需要一定的时间间隔，否则会输出"你已长按，请松手!"的提示。
