硬件说明：PORTB19=红灯；PORTB18= 绿灯；PORTB9= 蓝灯；

1. 工程概述
(1)本工程为KL25微控制器无操作系统应用工程
(2)本机KL25作为主机向与另一个作为从机的KL25相连接。本机复位后就不停的向从机写数据，
 而从机将本主机发送的数据通过串口打印出来

2. 软件环境
(1)Kinetis Design Studio IDE v3.0.0

3. 硬件环境
(1)KL25 SD13-01 EVB板
(2)MCU的串行口1与PC机的串行口相连                              
(3)将两块KL25板子的主机PTC9与从机PTC2、主机PTC8与从机PTC1，另外PTC9与PTC2、PTC8与PTC1分别
         通过电阻（推荐阻值为3.3K）上拉到+5V电源。注意：该驱动在3.3K电阻负载下开发，改
         变其负载可能需要改变MULT与ICR，即改变保持值SDA，否则可能通信不稳
	
4. 实验现象
(1)按本机的复位键通过串口发送字符串"This is i2c Test!"
(2)蓝灯亮暗交替，每变化一次向从机发送一位数据
注意：读者如需其他波特率，可自行查看寄存器进行配置
