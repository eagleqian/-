硬件说明：由于led数据位和片选位占用三色灯程序的引脚，所以本测试程序
                   停用三色小灯程序；


1. 工程概述
(1)本工程为KL25微控制器无操作系统应用工程，GPIO测试LED、LCD以及键盘程序

2. 软件环境
(1)KDS IDE v3.0.0.

3. 硬件环境
(1)KL25 SD13-01 EVB板
(2)对于键盘：PORT_A的2、5、1、4是行线，12、13、16、17是列线。
       对于LCD：  LCD数据位7~14引脚接PORT_D0~4和PORT_C的13,16,17。
           LCD控制位4~6引脚接PORT_D5，6，7
           LCD1~3引脚接电源，15引脚悬空，16引脚接地。
       对于LED: LED数据线接PORT_B的11,10,9,8,3,2,1,0；
                           位选口接PORT_B的19,18,17,16
	
4. 实验现象
(1)KL25 SD13-01 EVB板板上的LCD显示“Wait Receiving..Soochow 2016.04.”
(2)键盘按得数据显示在LCD上面。例如按键为9，LCD显示“The keyboard you just input is 9”
(3)LED上显示0235
